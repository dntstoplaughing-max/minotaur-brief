# Minotaur-Brief

A pre-registered benchmark for long-form federal-court advocacy.
Source: Mullins v. Duquesne Univ., No. 2:25-cv-01366 (W.D. Pa.).

## Privacy and pseudonymization

Third-party names appearing in the public docket are pseudonymized to role
labels in this dataset (and in the included model outputs), though unredacted
in the public record. Named defendants and the plaintiff (the first author)
are not pseudonymized; their identification is disclosed in the paper.
Pseudonymization pass v2, 2026-06-10.
